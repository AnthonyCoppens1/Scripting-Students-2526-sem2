# DSPSLovers
## simpleFunctions
- helloworld(): prints hello world
- helloname(name): prints hello followed by a given name 
- sum(x,y): calculates the sum of 2 given numbers
- average(x,y): calculates the average of 2 given numbers
- power(x,y): calculates the power of x to y

## TrueLove
- calculateTrueLove(name1, name2): calculates the true love % between 2 given names
- crappyLove(name1, name2): calculates a terrible version of the true love matching system
